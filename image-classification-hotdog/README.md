# Image Classification – Hotdog vs Not Hotdog

This project explores deep learning approaches for binary image classification using convolutional neural networks (CNNs).

The objective was to classify whether an image contains a hotdog or not using PyTorch-based models and transfer learning techniques.

## Project Highlights

* CNN image classification using PyTorch
* Transfer learning with EfficientNet-B0
* Batch normalization and data augmentation
* Early stopping and model evaluation
* Grad-CAM and SmoothGrad saliency maps
* Performance comparison between different architectures

## Results

* Baseline CNN achieved approximately 80.8% test accuracy
* Transfer learning with EfficientNet-B0 improved performance to approximately 95% accuracy
