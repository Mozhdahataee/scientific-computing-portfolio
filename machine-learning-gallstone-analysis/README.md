# Machine Learning – Gallstone Disease Analysis

This project explores machine learning methods for regression and classification analysis on a biomedical gallstone disease dataset.

The project investigates relationships between biological markers, inflammation, and gallstone disease using statistical and machine learning models.

## Project Highlights

* Ridge Regression
* Logistic Regression
* Artificial Neural Networks (ANN)
* k-Nearest Neighbors (KNN)
* Two-level cross-validation
* Statistical evaluation using McNemar’s test
* Feature importance and model interpretability

## Objectives

* Predict CRP (C-reactive protein) concentration using regression models
* Predict gallstone disease presence using classification models
* Compare model performance and generalization ability

## Results

* Logistic Regression achieved the best classification performance
* Ridge Regression produced interpretable and biologically meaningful coefficients
* Statistical evaluation demonstrated significant differences between classification models
